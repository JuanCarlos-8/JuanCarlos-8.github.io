HTML, o Lenguaje de Marcado de Hipertexto , es el lenguaje estándar para estructurar el contenido de las páginas web. Utiliza etiquetas para definir elementos como texto, imágenes, enlaces y más, permitiendo a los navegadores interpretar y mostrar la información en internet. HTML es fundamental para el desarrollo web, complementado por CSS para el diseño y JavaScript para la interactividad.

CSS, o Hojas de Estilo en Cascada, es un lenguaje de estilos que define la apariencia de los documentos HTML, como el color, la fuente, el tamaño, la posición y el diseño de una página web. Separa la estructura del contenido (HTML) de su presentación, lo que permite a los desarrolladores y diseñadores crear sitios web visualmente atractivos y consistentes, facilitando el mantenimiento y la flexibilidad del diseño.
